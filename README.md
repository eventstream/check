# @eventstream/check
Library for checking email, url and other

# Use Email check

for Node.js

```javascript
const isEmail = require('@eventstream/check').isEmail

isEmail('qwerty@gmail.com') // true
```

for React/Vue/Angular

```javascript
import { isEmail } from '@eventstream/check'

isEmail('qwerty@gmail.com') // true
```
